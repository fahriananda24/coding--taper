HTML

html adalah singkatan dari HyperText Markup Language dan merupakan bahasa markup standar yang digunakan untuk membuat struktur dan konten halaman web. HTML berfungsi sebagai kerangka dasar bagi sebuah halaman web, menentukan bagaimana teks, gambar, dan elemen-elemen lain ditampilkan

css

CSS, singkatan dari Cascading Style Sheets, adalah bahasa yang digunakan untuk mengatur tampilan dan gaya halaman web, seperti warna teks, font, tata letak, dan lainnya

JavaScript

JavaScript adalah bahasa pemrograman yang menambahkan interaktivitas ke situs web dan dapat menggunakannya untuk mengontrol hampir semua hal seperti validasi data formulir, fungsionalitas tombol, logika permainan, gaya dinamis, pembaruan animasi, dan banyak lagi.
